## 👋 About Me

사용자의 흐름을 먼저 생각하고, 기능을 끝까지 검증하는 프론트엔드 개발자 김우진입니다.  
React와 TypeScript 기반 프로젝트에서 API 연동, 인증 흐름, 상태 관리, 마이페이지, 검색/추천 UI를 구현해왔습니다.

- 사용자 경험을 해치지 않는 빈 상태, 로딩, 에러 흐름을 중요하게 봅니다.
- 기능 구현 후 lint/build와 브라우저 확인까지 이어지는 검증 습관을 갖고 있습니다.
- Codex, ChatGPT, Gemini, Manus 등 AI 도구를 요구사항 정리, 코드베이스 분석, 구현 방향 비교, 디버깅 보조에 목적별로 활용합니다.

## ✨ Highlight Experiences

- 🧩 **핵심 기능 구현**: 고객센터 챗봇, 소셜 로그인, 마이페이지 등 사용자가 직접 마주하는 주요 기능을 React와 TypeScript 기반으로 구현했습니다.
- 🏗️ **아키텍처 개선**: auth, matching, mypage 등 도메인 기반 폴더 구조를 설계하고 컴포넌트를 리팩토링해 기능 단위 유지보수성을 높였습니다.
- 🔐 **세션 안정성 개선**: Access Token 관리 정책을 sessionStorage 및 인메모리 중심으로 정리해 새로고침/탭 이동 상황에서 인증 흐름이 흔들리지 않도록 개선했습니다.

## 🤖 AI Agent Workflow

AI 도구를 단순 코드 생성 도구가 아니라, 문제 해결을 빠르게 검토하고 구현 품질을 높이는 보조 워크플로우로 사용합니다.

- Codex: 코드베이스 분석, 기능 구현 방향 비교, 리팩토링 후보 검토
- ChatGPT: 요구사항 정리, 문장화, 발표/이력서/README 구조 설계
- Gemini: 대안 비교, 긴 문맥 요약, 학습 내용 교차 확인
- Manus: 자료 정리와 반복 작업 보조
